01 · Hardware — Botões e Conectores

O KartBox tem três botões físicos na lateral do gabinete. Todos operam em lógica invertida (ativo em LOW, pull-up interno).

Botão 1
MODE
Alterna entre os modos de corrida. Só funciona com sessão parada.
tap → QUALY ↔ RACE
Botão 2
SET LINE
Grava a posição e direção GPS atual como linha de chegada.
tap → marca gate
Botão 3
RESET
Inicia uma nova sessão (tap). Encerra a sessão ativa (segurar).
tap → inicia
hold 1,5 s → encerra
Nota de montagem
Posicione o KartBox fixo no kart antes de marcar a linha — a direção (heading) do GPS no momento do SET LINE define o vetor de cruzamento. Se o dispositivo estiver solto ou girado, o gate vai detectar cruzamentos errados.

02 · Primeira vez: marcando a linha de chegada

  1. 1Ligue o KartBox e aguarde o ponto GPS na status bar ficar verde. Isso indica fix válido com pelo menos alguns satélites.
  2. 2Dirija o kart até a linha de chegada, posicionado como se fosse cruzá-la normalmente (mesma direção que você vai cruzar na prova).
  3. 3Pressione SET LINE uma vez. O KartBox congela latitude, longitude e heading nesse instante — esse é o gate.
  4. 4A partir daí, cada vez que o kart cruzar o raio configurado (padrão 15 m) na mesma direção, uma volta é registrada.
  5. 5O gate sobrevive a desligamentos — não precisa remarcar a cada sessão no mesmo kartódromo. Refaça o SET LINE só se mudar de pista.
Dica de precisão
Se o GPS demorar pra fixar (sinal fraco, garagem coberta), espere o indicador GPS 6+ antes de marcar. Com menos de 4 satélites o erro de posição pode fazer o gate não detectar cruzamentos corretamente.

03 · Tela principal — Aba CORRIDA

Essa é a tela que fica aberta durante a sessão. Toda a informação relevante está visível sem precisar rolar ou tocar em nada.

Aba CORRIDA — sessão ativa, volta 7, perdendo +0,13 s para o best

Barra de LED (Delta visual)

A barra de segmentos no topo funciona como shift-light de RPM — mas para o delta de tempo. Cada lado tem 8 LEDs, cada um representando ~187 ms de diferença (escala de 0 a 1.500 ms total). A barra inteira acesa = 1,5 s de diferença.

LEDs vermelhos (esquerda) acesos → perdendo tempo para o best
LEDs verdes (direita) acesos → ganhando tempo, melhor que o best
Todos apagados → primeira volta ou sem best ainda

Flash de final de volta

Ao cruzar a linha, o centro da tela pisca por 500 ms:

Dourado — novo recorde absoluto (best lap batido)
Verde — volta mais rápida que o melhor anterior (mas não o menor geral)
Vermelho — volta mais lenta que o best

04 · Histórico — Aba VOLTAS

Permite ler volta-a-volta de qualquer sessão gravada no cartão SD, sem precisar conectar o cabo.

Aba VOLTAS — sessão 20250620 com 5 voltas. Volta 3 é o best (fundo verde)

Como usar

  1. 1Toque em VOLTAS na barra inferior.
  2. 2O dropdown no topo lista todas as sessões salvas no SD, mais recente primeiro. Formato: AAAAMMDD_HHMMSS.csv.
  3. 3Toque no dropdown e selecione a sessão desejada — a lista de voltas carrega automaticamente.
  4. 4Se o SD foi modificado externamente (via WiFi ou USB), toque em atualizar para recarregar a lista.

Colunas da lista

ColunaDescrição
Volta NNúmero sequencial da volta na sessão
TempoDuração em segundos e milissegundos (ex: 1.334 = 1 min 33,4 s)
DeltaDiferença vs o best até aquele ponto da sessão. Verde = mais rápido, vermelho = mais lento
Vel maxPico de velocidade GPS registrado durante a volta
Destaque visual
A volta com menor tempo da sessão tem fundo verde e exibe BEST no lugar do delta — fácil de identificar com o rabo do olho.

Editar nome BLE / senha WiFi

  1. 1Role a aba CONFIG para baixo até ver os campos NOME BLE e SENHA WIFI.
  2. 2Toque no campo desejado — o teclado LVGL aparece sobreposto na parte inferior da tela.
  3. 3Edite o texto e pressione ✓ para salvar. O valor é gravado na NVS imediatamente.
  4. 4Pressione ✕ para cancelar sem salvar.

Apagar tudo (SD)

Ação destrutiva — sem recuperação
O botão vermelho apaga todos os arquivos CSV do cartão. Por segurança, exige dois toques: o primeiro muda o texto para "Toque de novo pra confirmar" e inicia um timer de 3 segundos. Se não confirmar nesse intervalo, o botão volta ao estado original automaticamente.

06 · Fluxo completo de sessão

  1. 1Defina o gate — posicione o kart na linha e pressione SET LINE. Só precisa fazer isso uma vez por pista.
  2. 2Inicie a sessão — pressione RESET uma vez. O ponto REC fica vermelho na status bar. Cada arquivo tem nome com data/hora local (ex: 20250620_103015.csv).
  3. 3Pilote — a cada cruzamento da linha o KartBox fecha a volta, atualiza a tela, pisca o overlay e grava a linha no CSV.
  4. 4Encerre — segure RESET por 1,5 s. O arquivo é fechado com segurança (flush + fclose). O ponto REC apaga.
  5. 5Leia os dados — use a aba VOLTAS para ver o resumo na tela, ou exporte via WiFi / USB para análise completa.
Sem cartão SD
O KartBox funciona mesmo sem SD — a tela e a telemetria BLE continuam normais. Só a gravação é desabilitada. O aviso aparece no log serial durante o boot.

07 · Telemetria BLE ao vivo

O KartBox anuncia continuamente via BLE (NimBLE) a 10 Hz. Um app no celular ou tablet pode receber velocidade, posição, volta atual e delta em tempo real — sem precisar tocar na tela do KartBox.

Status bar — sessão RACE ativa, BLE conectado
IndicadorEstado
BLE azulCliente conectado e recebendo telemetria
BLE cinzaAnunciando, sem cliente conectado
Nome do dispositivo
Por padrão o dispositivo aparece como KartBox. Mude na aba CONFIG → campo NOME BLE. A alteração só aparece após reiniciar o dispositivo (a NVS é gravada imediatamente, mas o anúncio BLE só muda no próximo boot).

08 · Exportar via WiFi

O KartBox cria um ponto de acesso WiFi sob demanda para servir os arquivos CSV. O WiFi não fica ligado o tempo todo — só quando você pede, para não disputar CPU e rádio com a gravação.

Aba CONFIG — WiFi export ativo. Informações de conexão aparecem na tela.
  1. 1Encerre a sessão (WiFi é bloqueado durante gravação para não afetar a performance).
  2. 2Na aba CONFIG, toque em Ativar WiFi export. O botão muda de texto e aparecem o SSID, senha e IP.
  3. 3Conecte o celular no WiFi cujo nome começa com KartBox-.
  4. 4Acesse http://192.168.4.1 no navegador para baixar os arquivos CSV.
  5. 5Quando terminar, toque em Desligar WiFi export. O AP também desliga sozinho após 5 minutos sem acesso (proteção automática).
Mudar a senha
Na aba CONFIG, role para baixo e edite o campo SENHA WIFI. O novo valor é gravado na NVS e usado na próxima vez que ativar o WiFi export. Mínimo de 8 caracteres (padrão WPA2).

09 · Modo pen drive (USB)

Transforma o KartBox em um pen drive — o SD aparece como volume no computador. Permite arrastar os CSVs sem precisar retirar o cartão do slot.

Exclusão mútua obrigatória
O barramento SDMMC só tem um dono por vez. Ao entrar no modo USB, o firmware desmonta completamente o SD antes de entregá-lo ao host. Toda gravação e leitura pelo firmware para enquanto o USB estiver ativo. Ao sair, o SD é remontado automaticamente.
  1. 1Conecte o cabo USB ao computador.
  2. 2Encerre qualquer sessão ativa (modo USB é bloqueado durante gravação).
  3. 3Na aba CONFIG, toque em Modo pen drive (USB). O SD aparece como volume removível no SO.
  4. 4Copie ou exclua os arquivos pelo computador normalmente.
  5. 5No computador, ejete o volume com segurança. Depois toque Encerrar modo pen drive na tela — o SD é remontado e o KartBox volta ao funcionamento normal.

10 · Modos QUALY vs CORRIDA

O botão MODE físico alterna entre os dois modos. Só pode ser mudado com sessão parada.

QUALY

O timing começa imediatamente ao cruzar a linha pela primeira vez. Ideal para treino livre onde você parte do pit sem largada formal.

CORRIDA (RACE)

O timing só começa quando o kart ultrapassa 5 km/h após cruzar a linha. Evita contar o tempo parado no grid esperando a largada.

O modo atual é exibido na pílula central da status bar:

QUALY
Fundo verde
RACE
Fundo vermelho

Análise externa
Com o CSV em mãos, ferramentas como RaceChronoPro, Python/pandas ou mesmo Excel permitem traçar mapa de pista (lat/lon), análise de setor, velocidade por trecho e comparação de voltas com granularidade de 100 ms.
