KARTBOX v2 — Editor de Pista

Gera arquivo .trk binário no mesmo layout de bytes que o firmware espera (track_manager.c). Marque a linha de chegada e, se quiser, até 2 setores.
Sua localização (GPS do celular) só funciona em "contexto seguro" (https, localhost ou arquivo local). Essa página foi aberta sem isso, então o botão "usar minha localização" pode falhar. Marcar no mapa manualmente e a sincronia com o kartbox continuam funcionando normal.

Sincronizar com o Kartbox (wifi)

Se o kartbox estiver no modo AP proprio (padrão): conecte o celular no wifi KartBox-XXXX antes de listar/enviar/apagar - endereço http://192.168.4.1.
Se o kartbox estiver no modo Cliente (conectado na sua rede, configurável na aba Config do equipamento): use http://kartbox.local com o celular na mesma rede - não precisa trocar de wifi nenhuma vez.
Nos dois casos dá pra manter essa aba aberta e só trocar/ajustar a rede na hora de sincronizar.
Formato do arquivo (112 bytes, little-endian, struct track_config_t):
offset 0: magic u32 (0x4B545231 "KTR1")
offset 4: name char[32]
offset 40: finish (24 bytes)
offset 64: setor 1 (24 bytes)
offset 88: setor 2 (24 bytes)
cada ponto: lat f64, lon f64, heading f32, is_set u8 (+3 pad)
